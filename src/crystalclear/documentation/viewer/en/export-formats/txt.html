Text Export

With the following properties you can configure the Text export.

Line Break

The parameter "newline" specifies the characters used for line breaks. By default, the server default value is used.

Page Break

The parameter "pagebreak" specifies the page delimiter for plain text format. By default, a line of '=' is used. If the last character is not a control character (lesser as white space) then a new line was added.

Codepage

The parameter "encoding" specifies the encoding for the txt file. By default, the server default encoding is used.
