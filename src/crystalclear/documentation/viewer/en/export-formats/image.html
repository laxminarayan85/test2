JPEG, PNG, BMP and GIF Export

With the following properties you can configure the JPEG, PNG, BMP and GIF export.

Image Size

The parameters "zoom", "width" and "height" specifies the size of the image to be created. You can only set one of it. Zoom can be set in percent. Width and height are set in pixel.

Background

Specifies the color of transparent parts of the image. The color is in ABGR format and can be set as decimal or hexdump number, e.g. 255 or 0xFF for the color red.
