CSV Export

Mit den folgenden Parametern kann man den CSV Export konfigurieren.

Spalten-Trennzeichen

Der Parameter "delimiter" spezifiziert das Spalten-Trennzeichen. Standardmäßig wird ein Komma verwendet.

Feste Breite

Der Parameter "colswidth" spezifiziert eine kommagetrennte Liste von Spaltenbreiten, z.B. 3,5,5 oder 5,5,,4. Die folgenden Standardwerte werden für Spalten verwendet, für die keine Breite angegeben wurde:

  • String: 8 Zeichen
  • Number: 10 Zeichen
  • Currency: 11 Zeichen
  • Boolean: 6 Zeichen
  • Date: 11 Zeichen
  • Time: 9 Zeichen
  • DateTime: 21 Zeichen

Wurde eine feste Breite angegeben, enthält die erzeugte CSV-Datei keine Spalten-Trennzeichen und Textbegrenzungszeichen.

Textbegrenzungszeichen

Der Parameter "quotechar" spezifiziert das oder die Zeichen, die als Textbegrenzungszeichen verwendet werden. Standardmäßig wird ein Anführungszeichen verwendet.

Codepage

Der Parameter "encoding" spezifiziert die Kodierung der CSV-Datei. Standardmäßig wird die Standard-Kodierung des Server verwendet.

Nur Daten exportieren

Spezifiziert, ob nur die Daten exportiert werden sollen. Die exportiere CSV-Datei enthält dann keine Textfelder und Layout-Informationen.

Mit Spaltennamen

Der Parameter "columnnames" spezifiziert, Spaltennamen mit exportiert werden sollen, d.h. ob die 1. Zeile der CSV-Datei die Spaltennamen enthält. Standardmäßig werden die Spaltennamen mit exportiert.
Dieser Parameter ist für den Daten-Export verfügbar.
