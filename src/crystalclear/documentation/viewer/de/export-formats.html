i-net Crystal-Clear

Exportformate

Mit i-net Crystal-Clear ist es möglich, Berichte in viele verschiedene Formate zu exportieren. Im Export-Dialog wählt man das Exportformat aus und kann es über formatspezifische Parameter konfigurieren.

Die folgenden Exportformate sind über Parameter konfigurierbar:

  • PDF
  • HTML
  • Postscript
  • MS Excel
  • CSV (Comma-Separated Values
  • Text
  • JPEG
  • PNG
  • BMP
  • GIF
